# MERN YouTube Watch Party

A real-time YouTube watch party application built using the MERN stack with synchronized playback, role-based permissions, and live room management using Socket.IO.

---

## Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- Socket.IO Client

### Backend
- Node.js
- Express.js
- Socket.IO
- MongoDB + Mongoose

---

## Features

- Create and join watch rooms
- Real-time synchronized playback
- Host / Moderator / Participant roles
- Role-based control system
- Change video for all participants instantly
- Live participant updates
- Remove participants from room
- Responsive minimal UI
- Production-ready CORS configuration
- Environment-based deployment setup
